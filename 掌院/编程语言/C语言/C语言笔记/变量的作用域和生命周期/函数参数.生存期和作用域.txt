1. int main(void){}  
  
   int 为函数返回值的类型 

2.return 

  当函数类型与返回值不一致的时候，返回值以函数类型为主。

  return 用于终止函数，返回值；当其后表达式为空时，只表示种植函数，不返回值。

3.一个程序必须有一个主函数，主函数可以调用普通函数，普通函数不可以调用主函数，普通函数之间，可以相互调用


4.函数的调用要放在定义的后面，要想不出错，需要声明函数。

  函数声明：告诉编译器代码是函数，以及形参和返回值的情况

5.调用函数时给的值与参数类型不匹配时c语言的最大漏洞，编译器总是悄悄帮你把类型转换好，但是多数情况下不是你所期望的

全局变量：没有初始化的全局变量会得到0值，函数内部与全局同名的变量，会将全局变量隐藏

static:特殊的全局变量，函数调用完会保持其值，只初始化一次，全局的生存期，本地的作用域
